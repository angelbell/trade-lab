# 仕様カード17 — イベント直後の短期モメンタム・スキャルプ（5分確認→建て→数分決済）

## 目的（ユーザーの再定義）
24h継続が消えるのは法則9で当たり前。**狙いは別の賭け**: FOMC等の発表直後、**最初の5分で方向を確認→その方向に建て→
数分（本命5分）で決済**。イベント・バーに短期モメンタム（次の5分が同方向に伸びる自己相関）があるか、
それが**往復コストを抜けるか**、**同時刻ランダムより強いか**を測る。[[feedback-low-tf-gamble-up]]（低TF・高頻度で増やす）
に合致。[[feedback-cost-after-edge]] 判定順=素の率×幅→偶然性→コスト→口座寄与。強度でなく「取る/見送る」の可否。

## 流用（車輪の再発明禁止）
- `scratchpad/fomc_event_study.py` / `event_kinetics.py` の関数（price_before・イベント読込・同時刻ランダムnull・
  ブロック/ブートストラップ）をそのまま import。**新規は「短期スキャルプの出口」だけ**。

## データ
- gold m5（`vantage_xauusd_m5.csv`, 2018-10-01以降=密化後）・BTC m5（`vantage_btcusd_m5.csv`, 2019-01-01以降）。
  **m1は不可**（goldは2025-12以降のみ・BTCは無し。カード15で確認）。時刻はブローカー時刻のまま。
- イベント = `data/ext_fomc_dates.csv`（71会合・dt_broker・**読み取り専用**）。CPI/NFPは保留（別台帳を渡せば流用可・カード15設計）。

## 機構（先読み厳禁）
- t0 = リリース時刻。P0 = t0直前確定足の終値。
- **確認窓 w_c = 5分**（発表後の最初のm5バー = t0を含む/直後のバー）。方向 **d = sign(close_{t0+5分} − P0)**。
  d=0（動かず）はスキップ。**建て = 確認バーの終値（t0+5分）で成行**（この価格は確認完了時点で既知＝先読み無し）。
- **保有 H ∈ {5,10,15,20,30,60}分** で決済（決済 = t0+5分+H の終値）。本命 H=5分。
- 参考: w_c=10分版も1回だけ（確認を1バー延ばすと follow-through が変わるか）。多重比較は増やさない。

## 計測（各 H で）
- 生の値幅 g = d·(P_exit − P_entry)。単位: **$/oz(gold)・$(BTC)** と **ATR(m5)比** の両方。中央値・p25・p75・std＋n
  [[feedback-prob-report-std-median]]。win% = P(g>0)。
- **コスト差引後**: gold 往復 $0.30（保守 $0.60も）、BTC 往復 $15（$10/$25も）を g から引く。**net の meanR相当・win%・
  P(net>0)・totで口座寄与**。5分の値幅がコストを抜けるかが生命線 [[feedback-edge-size-threshold]]（薄い日中は down-rank）。
- **follow-through**: Spearman(確認5分の動き, 保有中の動き d方向)＝モメンタムが正か（正なら"大きく動いたら乗る"が成立）。
- **null**: 同ブローカー時刻・非FOMC平日ランダム（3000回相当・カード14/15の機構流用）で同一手続き。
  実測の net-mean / win% / follow-through が null分布の何%ileか。**イベント特有か、ただの5分モメンタムかを分ける**。
- IS/OOS（前半/後半 or 年別）と --peryear 相当で時代分散（71会合を年で割ると薄いのは明記）。

## 報告
- 銘柄×H の表: gross中央値($・ATR比) / win% / **net-mean・P(net>0)・年totR相当** / null%ile。
- follow-through の Spearman と null%ile。w_c=10分版の一致/不一致。
- 平文結論: 「発表5分確認→5分保有」は **コスト後で+か / どの H が最良か / イベント特有か**。
  取れるなら可否ルール（この銘柄・この保有・このコスト前提で take）として出す。取れないなら病巣（コスト死/モメンタム無し/ランダム同等）を特定。

## 死に方（予想）
- **コスト死が本命**: 5分の値幅の中央値がコストに届かない（特にgoldの$0.30、BTCの$15）。
- モメンタム自体が null と同等（イベントでなく一般的な5分自己相関）＝イベント・エッジでない。
- 逆に即メンリバ（30%全戻し系）で d方向の follow-through が負なら「発表直後は乗るな、むしろ逆張り」の芽（別カードへ）。
- n=71・年8会合＝薄い。年totは会合数で振れる、を明記。
