## 2. 聖典 vs 実装の切り分け（監査の結論）

- **聖典側の問題**: (1) Draw on Liquidity は事前定義を持たない（事後の物語＝反証から構造的に
  逃げられる）。(2) 銘柄横断の普遍性の主張は実測と矛盾（忠実部品は EURUSD-long でだけ効く）。
  (3) ep41 部分利確はプロップ口座の生存則であり期待値最大化でない。(4) SMT＝装飾（残差はドル方向）。
- **実装側の問題**: (1) プール・モデルの2欠陥（一度タップで永久除外＝流動性は再充填されるのに／
  射程6×ATRを m15 ATR で測った単位ミスマッチ）→ **T_dyn の kill の射程は「この実装」までで、
  概念そのものはまだ**（下記 案2）。(2) FVG 閾値0.15 は旧入口の下で山型（プラトー無し）のまま
  新入口の下で再掃引されていない（案3）。(3) 母集団の結晶化は1通りだけ（ただし再採掘は DSR 壁）。
- **検証法の星印**: 旗艦の PF1.41 は逐次スイープの勝者を同じ21年上で組んだ合成物＝前向き期待は
  それ以下。12moブロックは16個しかなく小さい効果は検出できない。

## 3. 改善案（優先度順）

### 案1 【最優先・コスト≒0】 Pine に B案を並走させ、前向きで決める
- **機構**: 変換率解剖の最良セル＝入口0.25浅指値×損切−0.5ATR（E3×S2）は 2024-26 で唯一
  非負(+0.8R)。だが直近3年n≈40を見て選んだ後知恵であり、全史では現行 A案（CE×−0.1ATR）が
  最良（+87.9 vs +45.6）。**バックテストからはどちらが今後正しいか決められない** → 前向き比較へ。
- **実装**: Pine に入口/損切りを A/B 切替できる input を足し、同一チャートで両案を並走観察
  （事前登録された前向き比較＝後知恵コストゼロ）。
- **合格基準**: live-forward で B案が A案を勝率・PF で持続的に上回るか（判定時期はユーザー）。
- **どう死ぬか**: 両案とも赤字が続く＝素材の軟化が支配的 → 旗艦ごと畳む判断材料になる。

### 案2 【安い】 プール修正版 T_dyn の再測定 = kill の射程確定
- **機構**: 前回の「Draw on Liquidity は死」はプール実装の2欠陥込みの判定。
  (i) タップ後に価格が一定距離離れたら再武装、(ii) 射程を日足ATR基準に、(iii) ±数pipの
  等高値束をプールに——で再測定し、「聖典の概念の kill」か「実装の kill」かを台帳で確定させる。
- **事前登録**: おそらく「最近接プール≒PDH」に収束して現行と同値、が予想。
- **合格基準**: 現行 PDH 固定を12moブロックで安定して上回ること（前回と同じ掟）。
- **どう死ぬか**: PDH と同値に収束（→ PDH は「常に近くにある代用磁石」だったと確定、概念ごと閉じる）。

### 案3 【安い】 FVG 閾値の再掃引（旗艦の内部整合性の掃除）
- **機構**: fvg_min_atr=0.15 は旧入口(0.25)の下で山型の頂点（37.5→81.7→32.3%ile）。
  現行入口(FVG-CE)・出口(PDH)の下で {0(存在のみ), 0.10, 0.15, 0.20, 0.25} を再掃引。
- **合格基準**: 台地なら安心して凍結。山のままなら「存在のみ」へ緩めるか星印を付ける。
- **どう死ぬか**: 死なない（掃除であって新エッジ探しではない）。

### 案4 【保留・DSR 壁の警告付き】 素材が強い他ペアへの獲り方の移植
- 素材診断で GBPUSD/USDCAD の 2024-26 素材は歴史比最良水準。ただし忠実部品（CE入口・PDH）は
  他ペアで死んだ実績があり（法則2）、新しい組み合わせを掘るのは新規母集団の採掘＝DSR 壁に
  正面衝突する（v3/v4 で2回確認済みの構造）。**着手するなら「掘る」のでなく既存6セル分解を
  そのまま GBP/CAD に流す1回だけ**とし、多重比較を増やさない。

## 4. やらないこと（再テスト禁止・台帳照合済み）

- 部分利確+建値（ep41）——勝率は買えるが総利益の9割が対価。心理は賭け率ダイヤルで買う（法則4）。
- 週足MAレジームゲート／週足HHゲート——プラトー無しの蜃気楼。
- ドロー方向ゲート(G_dir)・ディスカウント限定(G_disc)——12moブロックで死（現行プール実装の下で）。
- SMT / DXY マスターバイアス——装飾。残差はドル方向ベータ。
- 深い OTE(62-79%)・近い目標(asiaH/swingH)・日足バイアス(トレンド解釈)——旧セッションで死亡済み。
- 新規 FX 母集団の再採掘——掘るたび DSR 壁（層を足すこと自体が試行）。

## 5. 物差し（この線の検証で使うもの）

- 勝率を前面に PF・トレード解像度 maxDD・年別を併記（単独システムの物差し）。
- DSR は棄却の門でなくサイズ入力。局所最適の排除は巡回ブロック・ブートストラップ(1/3/6/12mo)＋時期分散。
- 勝率上昇は機構的に保証されうる（部分利確・玉数間引き）ので単独では合格にしない。
- 決着は live-forward（既に走っている。再提案しない）。
