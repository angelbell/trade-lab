# 仕様カード13 — 時間帯・曜日・FOMC の文脈層別（btc15m_A ＋ gold15m・強度部品の候補）

## 目的
両レッグの発火トレードを「いつ建ったか」の文脈で層別し、EVが並ぶかを測る（単一変数の情報測定・
[[feedback-standalone-before-combination]]）。出力は**強度の段**（take/skip や強弱の読み）であり、
サイズ倍率は出さない [[feedback-fixed-lot-no-sizing]]。ユーザーの問い「FOMCのときはどうする」に答える。

## 前提（既存台帳との整合・再導出禁止）
- **gold15m のセッションスキップは禁止が確定済み**（捨てる窓が黒字・CLAUDE.md）。∴「窓を捨てろ」という結論は
  出さない。窓ごとの強弱（強い窓/弱い窓）として読む。矛盾する結果が出たら既存台帳と突き合わせて報告。
- btc15m_A の時間帯・曜日・FOMC は未検証（台帳に無し）。

## 対象レッグ（照合済み再構築を流用）
- btc15m_A: `scratchpad/strength_btc15mA.py` の経路（照合ゲート4本PASS・n=229・素netR）。
- gold15m: `scratchpad/strength_gateslope_generalize.py` の build_gold15m（照合PASS・n=325・netR 0.3/risk込み）。
- 時刻は**約定足（fill bar）の時刻**を主とする（＝実際に建つ瞬間。指値は事前に置くが、FOMC前に取り消す運用が
  可能なので約定時刻の文脈が実運用に対応）。参考でシグナル足時刻の層別も併記（安ければ）。
- **時刻はブローカー時刻（EET/EEST）**。データはそのままブローカー時刻なので変換不要だが、FOMC だけ ET→ブローカーの
  tz 変換が必須（下記）。

## 変数（各単体で）
1. **時間帯バケット（粗く4つ・ブローカー時刻）**: アジア(01-08) / ロンドン(08-15) / NY(15-22) / 深夜(22-01)。
   境界はキルゾーン慣行に合わせ、境界±1hのズレで結果が変わらないか1回だけ確認（多重比較を増やさない）。
2. **曜日**: 月〜金（BTCは土日も建ちうる→7つ、nを明記）。
3. **FOMC**: 開催日フラグ。
   - データ: 連銀公式 (federalreserve.gov のFOMCカレンダー・歴史資料ページ) から **2018〜2026 の会合日**（決定声明日
     =2日目）を取得し `data/ext_fomc_dates.csv` に保存（**決定論的な正典・Waybackは使わない**。案2の教訓）。
     既存 `data/ext_econ_calendar.csv` に FOMC 48行(2021-2026)が残っているので突き合わせ検証に使う。
   - 声明 14:00 ET → **tz変換必須**: ET→Europe/Riga（ブローカー時刻）。CLAUDE.md の先読み罠（素で突き合わせない）。
   - フラグ定義: (a) 約定が声明当日（ブローカー日付）(b) 約定が声明前24h以内 (c) 声明後24h以内、の3層。nが薄いのは
     承知で、meanR/win%/PF を n 明記で出す（無理に有意化しない）。
4. 参考: 3変数のうち候補（差が出たもの）だけ、巡回ブロック・ブートストラップ(1/3/6/12mo,3000回)で
   バケット間ギャップのCIを出す。**全バケットに検定を乱発しない**（多重比較・Bonferroni注記）。

## 報告
レッグ×変数ごとの層別表: n / n年 / win% / PF / meanR。差が出た候補のみブロックCI＋年別。
FOMC は「当日/前24h/後24h vs 平常」の表。結論は「強度の段としてどう読むか」（例: FOMC当日の約定は弱段/平常/強段）。
既存のセッションスキップ禁止と矛盾しないかの突き合わせも明記。
scratchpad/context_time_fomc.py。実行 .venv/bin/python。--smoke は直近2年。

## 死に方（予想）
- 時間帯・曜日: バケットを切れば凸凹は必ず出る→ブロックCIで潰れる（偽の凸凹）が本命。
- FOMC: n が薄すぎて判定不能（btc15m_A 年30本 × FOMC 年8回 → 重なりは数本）。その場合「読めるほどの標本が無い」
  と正直に書く。gold15m のほうが n=325 で読める可能性がやや高い。
- 生き残るとしたら「gold15m の FOMC 前後」（金は金利イベント感応度が高い）が相対的に有望。
