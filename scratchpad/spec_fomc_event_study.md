# 仕様カード14 — FOMC/重要イベントの価格挙動スタディ（継続 vs 全戻し・変動率で層別）

## 目的（ユーザーの問い）
FOMC等の重要イベントで価格が何をするかを、レッグ・エントリー無しで純粋に測る:
- ブレイクアウトして**そのまま抜ける（継続）**のか、
- **次の日には価格がほぼ100%戻る（全戻し）**のか。
- 主要な計測は **変動率（初動の大きさ）** と、**各時間ごと（最大24hまで）の最大戻り値**。
- 仮説: **変動率が低い場合はすぐ戻る**（小さい初動＝全戻し、大きい初動＝継続、を検定）。
これは強度部品でもレッグでもなく、価格挙動の記述＋null照合 [[feedback-prob-report-std-median]]。

## イベント（決定論カノン・読み取り専用）
- `data/ext_fomc_dates.csv`（71会合・2018-2026・dt_utc/dt_broker列）。**このファイルは読むだけ。書き込み禁止**
  （別の計測係が触っている可能性。レース回避）。dt_broker = 声明リリース時刻（ブローカー時刻）。
- 重要イベントの決定論セットは現状FOMCのみ（NFP/CPIは未整備）。今回はFOMCで確定させ、拡張は生き残ってから。

## 銘柄・TF
- gold m15（`data/vantage_xauusd_m15.csv`）と BTC m15（`data/vantage_btcusd_m15.csv`）＝ユーザーが実際に張る2市場。
- 頑健性の横目に 1h も1回だけ（m15の結論が1hで裏返らないか）。時刻はブローカー時刻のまま（tz変換不要）。

## 定義（先読み禁止・窓の分離を厳守）
- **t0 = リリース時刻**（dt_broker）。**アンカー P0 = リリース直前に確定した足の終値**（イベント足を見ない）。
- **初動窓 w0 = リリース後1h**（m15で4本）。方向 d = sign(P_{t0+1h} − P0)。初動の大きさ **I = |P_{t0+1h} − P0|**。
- **変動率（層別変数）= I を正規化**した量:
  - 生（$ / oz または $）
  - I / ATR14(m15, リリース直前)  … 主たる正規化
  - I / 直近実現ボラ（例: リリース前20本の終値変化のstd）… 副
- **地平線 h ∈ {1, 2, 4, 6, 8, 12, 18, 24}h**（t0起点）。

## 各イベント×hの計測量（方向 d で符号化）
1. **retain_h = d·(P_h − P0) / I** … 初動のうち残った割合（1=完全継続 / 0=行って来い / <0=反転して初値を越えて戻る）。
2. **round_trip_h = 1 − retain_h** … 戻った割合（≈1.0 が「ほぼ100%戻る」）。**next-day = round_trip_24h**。
3. **maxext_h = max_{[t0,t0+h]} d·(P−P0) / I** … 継続の最大到達（>1 = 初動を超えて伸びた）。
4. **maxret_h = 最大戻り値** = [t0,t0+h] の走行中、favの走行ピークからの最大の押し戻し / I
   （ピーク → その後の最安（d方向）までの下げ幅を I で割る。100% = ピークから初値まで丸ごと戻した、>100% = 初値を突破して反転）。
5. **P(全戻し) = round_trip_h ≥ 1.0 になった割合**（h=6,12,24 で）。

## 変動率での層別（仮説の検定）
- I/ATR14 の三分位（小/中/大）ごとに retain_24h・round_trip_24h・maxext・maxret の中央値/p25/p75/std。
- **Spearman(I/ATR, retain_24h)**（正 = 大きい初動ほど継続＝ユーザー仮説と整合、負 = 逆）。n明記。
- 「小さい初動はすぐ戻る」を retain の時間発展（1h→24h）で小群 vs 大群を並べて見せる。

## NULL（最重要・FOMCは本当に特別か）
- **同時刻・非イベント日ランダム**: 各FOMCと同じブローカー時計時刻に、FOMCでない平日をランダムに選んで
  同一手続きで測る（多数ドロー・3000回相当）。retain_h / round_trip_h / maxret_h の分布を実測FOMCと比較。
  **FOMC ≈ 同時刻ランダムなら「大きく動いて一部戻る」は価格の一般性でありFOMC効果でない**、を明示。
  実測FOMCが null分布の何%ileかを各指標で出す。

## 報告
- 銘柄×h の表: retain / round_trip / maxext / maxret を **中央値・p25・p75・std**（歪むので分位必須）＋ n。
- 変動率三分位表＋Spearman。FOMC vs null の%ile。1h横目の一致/不一致。
- 結論を平文で: 「FOMC後は継続寄りか全戻し寄りか」「変動率が低いと戻るは正しいか」「運用示唆＝強度の段としてどう読むか」。
- gold15m のセッションスキップ禁止は既存確定。ここは価格挙動の記述なので窓の取捨結論は出さない。

## 死に方（予想）
- FOMC ≈ 同時刻ランダム（イベント効果が薄い or サンプル71では検出不能）が本命。
- 変動率仮説は「小さい初動＝そもそも動いてない＝戻る余地も小さい」で自明に見えるだけの恐れ（maxret を I で正規化して潰す）。
- gold は金利感応で BTC より継続/反転の色が出る可能性が相対的に高い。
